# Image Cutter

Outil web pour découper des images en tuiles de taille définie.

## Fonctionnalités

- Découpe d'image en tuiles personnalisables (largeur x hauteur)
- Conservation ou suppression des zones vides
- Mode chevauchement 50%
- Téléchargement individuel ou par archive ZIP
- Mode sombre/clair
- Interface FR/EN
- Design responsive

## Utilisation

1. Sélectionner une image
2. Définir la taille des tuiles
3. Choisir les options de découpe
4. Télécharger le résultat

# Image Cutter

Web tool to slice images into tiles of defined size.

## Features

- Custom tile size slicing (width x height)
- Keep or remove empty areas
- 50% overlap mode
- Individual or ZIP download
- Dark/light mode
- FR/EN interface
- Responsive design

## Usage

1. Select an image
2. Define tile size
3. Choose slicing options
4. Download result
